# duplicate_database_mongodb

### script duplicate database mongodb to mongodb
### create file `.env`

```
MONGO_SOURCE_URL=
MONGO_DEST_URL=
SOURCE_DB=
DEST_DB=
```

### how to running 
```
git clone https://github.com/badoss/duplicate_database_mongodb.git
cd ./duplicate_database_mongodb
npm i
node dumpmongodb.js
```
